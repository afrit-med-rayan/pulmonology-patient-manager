📋 COMMENT UTILISER CE SYSTÈME (TRÈS SIMPLE!)

1. ✅ DOUBLE-CLIQUEZ sur "Launch_Patient_Manager.bat"
2. ✅ ATTENDEZ que le navigateur s'ouvre automatiquement
3. ✅ COMMENCEZ à utiliser le système de gestion des patients!

================================================================
🔧 SI ÇA NE FONCTIONNE PAS:
================================================================

❌ Erreur "Python non trouvé":
   → Installez Python depuis: https://python.org
   → Cochez "Add Python to PATH" pendant l'installation

❌ Le navigateur ne s'ouvre pas:
   → Ouvrez manuellement: http://localhost:8000/complete-patient-system.html

❌ Port déjà utilisé:
   → Fermez d'autres programmes qui utilisent le port 8000
   → Ou redémarrez votre ordinateur

🎯 FONCTIONNALITÉS DU SYSTÈME:
================================================================

📊 TABLEAU DE BORD
   → Vue d'ensemble des patients et statistiques

👤 CRÉER UN PATIENT
   → Ajouter de nouveaux patients avec toutes les informations

🔍 RECHERCHER
   → Trouver rapidement des patients par nom, téléphone, etc.

📋 LISTE DES PATIENTS
   → Voir tous les patients enregistrés

📝 VISITES
   → Gérer les visites avec les nouveaux champs:
     • CONSULTATION (observations cliniques, symptômes, notes)
     • BILLAN (remarques sur le bilan)
     • CAT (médicaments prescrits)
     • EXAMEN CLINIQUE (remarques sur l'examen)
